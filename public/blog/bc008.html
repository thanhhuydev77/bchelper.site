Visual Branding: Adding Logos to Shipping Agents & Exporting to Excel

Background

In a manufacturing environment, logistics involves managing numerous shipping partners. However, the default Business Central interface for Shipping Agents is strictly text-based, which can be inefficient for quick identification during high-pressure warehouse operations.

The Problem The standard Shipping Agent page lacks visual cues, making it "boring" and prone to selection errors. Furthermore, warehouse teams often require Excel exports that include these logos to help external drivers and coordinators identify the correct carriers at a glance.

The Solution

To modernize the shipping management experience, we implement a three-tier solution:

  • Schema Update: Add a "Logo" field using the Media data type for better performance.
  • UI Enhancement: Implement a FactBox to manage and display images.
  • Reporting: Create an SSRS Report that streams these images into professional Excel or PDF exports.

Technical Implementation

Media InStream FactBox SSRS

Step 1: Create Table Extension

We use the Media type instead of Blob. This is the modern standard for BC Cloud, as it handles scaling and performance more efficiently. We also add a fieldgroup for Brick view to enable tile visualization.

ShippingAgentExt.Table.al
tableextension 70000 "ALE Shipping Agent" extends "Shipping Agent"
{
    fields
    {
        field(70000; "ALE Logo"; Media)
        {
            Caption = 'Picture';
        }
    }
    fieldgroups
    {
        addlast(Brick; Code, Name, "ALE Logo") { }
    }
}

Step 2: Create Page CardPart for the Logo

This part acts as the container for the image and provides actions to Import, Export, or Delete the logo using the Media management methods.

ALEShippingAgentLogo.Page.al
page 70000 "ALE Shipping Agent Logo"
{
    Caption = 'Shipping Agent Logo';
    PageType = CardPart;
    SourceTable = "Shipping Agent";

    layout
    {
        area(content)
        {
            field(Picture; Rec."ALE Logo")
            {
                ApplicationArea = All;
                ShowCaption = false;
                ToolTip = 'Specifies the picture that has been inserted for the item.';
            }
        }
    }

    actions
    {
        area(processing)
        {
            fileuploadaction(ImportPicture)
            {
                ApplicationArea = All;
                AllowMultipleFiles = false;
                AllowedFileExtensions = '.jpg', '.jpeg', '.png';
                Caption = 'Import';
                Image = Import;
                ToolTip = 'Import a picture file.';

                trigger OnAction(Files: List of [FileUpload])
                var
                    Instream: InStream;
                begin
                    if Files.Count = 0 then exit;
                    Files.Get(1).CreateInStream(Instream);
                    Rec."ALE Logo".ImportStream(Instream, Files.Get(1).FileName);
                    Rec.Modify();
                end;
            }
            action(ExportFile)
            {
                ApplicationArea = All;
                Caption = 'Export';
                Image = Export;
                ToolTip = 'Export the picture to a file.';
                Enabled = HadContent;
                trigger OnAction()
                var
                    TenantMedia: Record "Tenant Media";
                    Instream: Instream;
                    FileName: Text;
                begin
                    Rec.TestField(Code);

                    TenantMedia.Get(Rec."ALE Logo".MediaId);
                    TenantMedia.CalcFields(Content);
                    TenantMedia.Content.CreateInStream(Instream);
                    FileName := TenantMedia.Description;
                    DownloadFromStream(Instream, '', '', '', FileName);
                end;
            }
            action(DeletePicture)
            {
                ApplicationArea = All;
                Caption = 'Delete';
                Image = Delete;
                ToolTip = 'Delete the record.';
                Enabled = HadContent;
                trigger OnAction()
                begin
                    DeleteItemPicture();
                end;
            }
        }
    }

    trigger OnAfterGetRecord()
    begin
        HadContent := Rec."ALE Logo".HasValue();
    end;

    procedure DeleteItemPicture()
    var
        DeleteImageQst: Label 'Are you sure you want to delete the picture?';
    begin
        Rec.TestField(Code);

        if not Confirm(DeleteImageQst) then
            exit;

        Clear(Rec."ALE Logo");
        Rec.Modify(true);
    end;

    var
        HadContent: Boolean;
}

Step 3: Extend Shipping Agent Page (FactBox)

Inject the CardPart into the FactBox area so users can see the logo immediately upon selecting a record in the list.

pageextension 70005 "ALE Shipping Agents" extends "Shipping Agents"
{
    layout
    {
        addlast(factboxes)
        {
            part(Image; "ALE Shipping Agent Logo")
            {
                ApplicationArea = All;
                SubPageLink = Code = field(Code);
            }
        }
    }
}

Step 4: SSRS Report Development

To show the logo in an Excel/PDF report, you must use InStream to pass the media data to the layout. In the RDLC layout, set the image source to Database and the MIME type to image/png or image/jpeg.

ALEShippingAgents.Report.al
report 70005 "ALEShipping Agents"
{
    ApplicationArea = All;
    Caption = 'Shipping Agents';
    UsageCategory = ReportsAndAnalysis;
    DefaultRenderingLayout = RDLC;

    dataset
    {
        dataitem(ShippingAgent; "Shipping Agent")
        {
            column("Code"; "Code") { }
            column(Name; Name) { }
            column(AccountNo; "Account No.") { }
            column(Picture; gTenantMedia.Content) { }
            trigger OnAfterGetRecord()
            begin
                Clear(gTenantMedia);
                if not ShippingAgent."ALE Logo".HasValue() then
                    exit;

                gTenantMedia.Get(ShippingAgent."ALE Logo".MediaId);
                gTenantMedia.CalcFields(Content);
            end;
        }
    }
    rendering
    {
        layout(RDLC)
        {
            Type = RDLC;
            LayoutFile = '.\Layout\ALEShippingAgents.rdlc';
        }
    }

    var
        gTenantMedia: Record "Tenant Media";
}

Pros & Benefits

  • Cloud Compatibility: Works seamlessly on SaaS and On-Premise environments.
  • Improved Identification: Visual logos allow for split-second recognition, reducing human error.
  • FactBox Utility: Keeps the main list clean while providing high-value data in the side pane.
